# Photo Editor JavaScript

## Features
- [x] Cropping 
- [x] Zooming
- [x] Rotate image
- [x] Flip vertical
- [x] Flip horizontal
- [x] Add blur filter 
- [x] Add brightness filter 
- [x] Add contrast filter 
- [x] Save and download edited image to your PC
